## PROJECT_agent-hub

[ES]
title = Agent Hub — Interfaz de Agentes IA sin Fricción
problem = El cuello de botella no era construir agentes de IA — era que las personas no técnicas no podían acceder a ellos sin fricción. La tecnología existía; la interfaz humana, no.
built = Definí la visión del producto, construí y curé más de 10 agentes, y diseñé una interfaz que elimina la fricción de acceso para usuarios no técnicos. Lideré el backlog completo y llevé el proyecto a pitch externo.
impact = De herramienta interna a propuesta con potencial comercial. Agent Hub cambió la conversación de "tenemos agentes" a "cualquiera puede usarlos".

[EN]
title = Agent Hub — Friction Reduction AI Agent Interface
problem = The bottleneck wasn't building AI agents — it was that non-technical users couldn't access them without friction. The technology existed; the human interface didn't.
built = Defined the product vision, built and curated 10+ agents, and designed an interface that eliminates access friction for non-technical users. Led the full backlog and took the project to external pitch.
impact = From internal tool to a commercially viable proposal. Agent Hub shifted the conversation from "we have agents" to "anyone can use them".

[ES]
